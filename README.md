# E-Commerce Front-End Website

This is a front-end e-commerce web application developed using **HTML, CSS, and JavaScript**.  
The project provides a simple and interactive online shopping experience with dynamic cart functionality.

## Features
- Product listing page
- Product details page
- Add products to shopping cart
- Remove products from cart
- Update product quantities
- Dynamic cart item count
- Real-time cart total price calculation
- handling form by formspree and testing Email Js too
- Buidling payment UI 

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Functionality
JavaScript is used to handle all cart operations, including adding items, removing items, updating quantities, and calculating totals dynamically without page reloads.
